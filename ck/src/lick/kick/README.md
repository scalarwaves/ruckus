lick.kick
---

How I use [LiCK](https://github.com/heuermh/lick) in [ChucK](http://chuck.cs.princeton.edu/):

Clone LiCK from Github

    $ git clone https://github.com/heuermh/lick
    $ cd lick

Then, in one terminal

    $ chuck --loop

In another terminal

    $ chuck + import.ck
    $ chuck + examples/kicksExample.ck
